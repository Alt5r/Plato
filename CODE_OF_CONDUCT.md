# Code of Conduct

## Our Pledge

We want participation in PlaTo to be respectful, constructive, and inclusive for everyone.

## Expected Behavior

- be respectful and professional
- assume good intent, but be clear and direct
- focus feedback on the work, not the person
- avoid harassment, discrimination, and personal attacks

## Unacceptable Behavior

- harassment or abusive language
- discriminatory or exclusionary conduct
- sustained disruptive behavior
- publishing private information about others without permission

## Reporting

If you experience or observe unacceptable behavior, report it privately through GitHub where
possible.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, or
other contributions that do not align with this code of conduct.
