# Contributing

Thanks for contributing to PlaTo.

## Getting Started

- read [README.md](README.md)
- review [SECURITY.md](SECURITY.md)
- review [docs/threat-model.md](docs/threat-model.md)

## Development

- make focused changes
- add or update tests for behavior changes
- run the test suite before submitting:

```bash
npm test
```

## Pull Requests

- describe what changed and why
- mention any security or runtime implications
- keep PRs scoped to one problem where possible

## Security Issues

If you believe you found a security issue, follow [SECURITY.md](SECURITY.md) instead of opening a
public issue first.
